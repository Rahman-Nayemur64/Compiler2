Mini Compiler Project (Python)
--------------------------------
This project implements a simple compiler with all major phases:
1. Lexical Analysis
2. Syntax Analysis
3. Semantic Analysis
4. Intermediate Code Generation
5. Optimization
6. Code Generation
7. Execution (via Virtual Machine)

Run with:
    python main.py

Example language syntax:
    a = 5;
    b = a + 3;
    c = b * 2;
